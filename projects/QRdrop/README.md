# QRdrop

## 介绍
QRdrop 是一个创新的 Web3 空投解决方案，通过二维码让代币分发变得简单高效。用户只需扫描二维码即可快速领取代币，特别适合会议、活动、社交聚会等线下场景。

## 主要特性

- 🎯 **简单易用**：扫码即可领取，无需复杂操作
- 🔒 **安全可靠**：基于智能合约的链上空投机制
- ⚡ **高效快捷**：批量生成二维码，快速分发代币
- 🌐 **多链支持**：支持多个区块链网络
- 📱 **移动友好**：优化的移动端体验

## 使用场景

- 会议和活动的参与奖励
- 社区线下聚会的代币分发
- 产品推广和市场营销活动
- NFT 项目的白名单发放
- 教育培训的激励机制

## 技术架构

- **前端**：React + ether.js
- **智能合约**：Solidity
- **支持网络**：Ethereum、BSC 等
